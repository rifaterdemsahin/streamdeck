üìã Prerequisites

  • Stream Deck software v6.0 or later

  • DaVinci Resolve Stream Deck plugin (from the Elgato Store or Marketplace)
  • A project with clips on the timeline to color

üõ†Ô∏è Steps to Set Up

Phase 1: Install the DaVinci Resolve Plugin

  1. 1
    Open Stream Deck Store
    In the Stream Deck software, click More Actions... or the Store icon. Search for DaVinci Resolve.
  2. 2
    Install the Plugin
    Install the official DaVinci Resolve plugin. It adds Resolve-specific actions including clip color, timeline navigation, and page switching.
  3. 3
    Restart Stream Deck Software
    After installation, restart the Stream Deck software to ensure the plugin loads correctly.

Phase 2: Configure Knob Click for Clip Colors

  1. 4
    Select a Dial Slot
    In the Stream Deck software, click on one of the 4 dial/knob slots at the bottom of the Stream Deck + layout.
  2. 5
    Assign the Click Action
    Under the "Press" (click) action for the dial, drag a Hotkey action or the DaVinci Resolve Clip Color action. Configure it to send the keyboard shortcut that sets the desired clip color in Resolve.
  3. 6
    Set the DaVinci Resolve Hotkey
    In DaVinci Resolve, go to DaVinci Resolve ‚Üí Keyboard Customization. Search for "Clip Color" and assign a unique hotkey to each color (e.g., Ctrl+Shift+1 for Red, Ctrl+Shift+2 for Orange, etc.).
  4. 7
    Map Hotkey to Knob Click
    Back in Stream Deck, set the knob click to send the matching hotkey. For example: Knob 1 click ‚Üí Ctrl+Shift+1 ‚Üí Red clip color.

Phase 3: Set Up All Four Knobs

  1. 8
    Assign Colors to Each Knob
    Configure all 4 knobs with your most-used clip colors:
    ‚Ä¢ Knob 1 Click ‚Üí üî¥ Red (needs fix)
    ‚Ä¢ Knob 2 Click ‚Üí üü¢ Green (approved)
    ‚Ä¢ Knob 3 Click ‚Üí üîµ Blue (audio/music)
    ‚Ä¢ Knob 4 Click ‚Üí üü° Yellow (review needed)
  2. 9
    Configure Rotate Actions
    Optionally, set the rotate action for each knob to something complementary ‚Äî e.g., timeline scrub, volume adjust, or zoom.
  3. 10
    Set Touch Strip Labels
    Customize the touch strip display between the dials to show which color each knob is mapped to ‚Äî the LCD strip shows the current assignment visually.
  4. 11
    Test in DaVinci Resolve
    Open a project, select a clip on the timeline, and click each knob. The selected clip should change to the assigned color instantly.

üé¨ Editing Workflow

Select a clip ‚Üí Click the knob ‚Üí Clip is colored instantly. No right-click menus, no keyboard combos ‚Äî just a satisfying physical click. Use Rotate to scrub to the next clip, then Click again to color it. You'll fly through timeline organization.
