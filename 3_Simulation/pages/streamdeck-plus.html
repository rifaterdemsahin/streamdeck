  • 8 LCD buttons
  • 4 rotary encoders with haptic
  • Touch strip with LCD
  • Best for: analog controls, fine adjustments

üí° Best Setup: Use the Stream Deck XL for buttons, shortcuts, and macros ‚Äî and the Stream Deck+ for rotary controls, audio levels, and fine-tuning. Together they cover both digital (on/off) and analog (continuous) interaction.

üöÄ Getting Started

Set up your Stream Deck+ for DaVinci Resolve in 5 minutes:

  1. 1
    Install Stream Deck Software
    Download from elgato.com/downloads ‚Äî version 6.0 or later required for SD+ features.
  2. 2
    Install the DaVinci Resolve Plugin
    In the Stream Deck Store, search for "DaVinci Resolve" and install the official plugin.
  3. 3
    Create an SD+ Profile
    Create a new profile in the Stream Deck software for your Stream Deck+ device. Assign it to DaVinci Resolve.
  4. 4
    Map Buttons & Knobs
    Assign edit actions to the 8 LCD keys and configure the 4 knobs for your workflow (see knob layouts above).
  5. 5
    Test & Refine
    Open DaVinci Resolve, test each button and knob, adjust haptic strength in Stream Deck settings. Iterate until it feels right.
