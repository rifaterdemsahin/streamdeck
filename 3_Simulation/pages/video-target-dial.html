What is the Video Target Dial?

The video target dial (or video track targeting) determines which video track in your timeline will receive incoming clips when you perform insert or overwrite edits. It's essential for multi-track editing, allowing you to direct your edits to specific tracks without manually selecting them.

üé¨ Common Use Cases

1. Multi-Camera Editing

Scenario: You have 3 camera angles on V1, V2, and V3

Workflow:

  1. Place all camera angles synced on separate tracks
  2. Target V4 for your edit track
  3. Press numbers 1-3 to switch camera angles
  4. Edits automatically go to V4, keeping originals intact

2. B-Roll Over Interview

Scenario: Interview on V1, want to add B-roll on V2

Workflow:

  1. Place interview on V1
  2. Target V2 (cycle up with Alt/Opt + Up)
  3. Disable A1 targeting (only video will be placed)
  4. Insert B-roll clips - they go to V2, audio stays on V1

3. Graphics and Text Overlays

Scenario: Adding titles and graphics over edit

Workflow:

  1. Reserve V3+ for graphics layers
  2. Target V3 when adding titles
  3. Main edit on V1-V2 remains untouched
  4. Easy to adjust graphics without affecting main edit

4. Audio Stem Editing

Scenario: Organizing dialogue, music, and SFX

Workflow:

  1. A1-A2: Dialogue
  2. A3-A4: Music
  3. A5-A6: Sound effects
  4. Target appropriate tracks when editing each type
  5. Keeps audio organized for mixing

üí° Pro Tips

Track Targeting Best Practices:

  • Establish a track plan: Decide which tracks are for what before you start editing
  • Use color coding: Assign colors to tracks (right-click track header)
  • Lock completed tracks: Prevent accidental edits to finished work
  • Name your tracks: Right-click track header to add descriptive names
  • Visual check before editing: Always verify target indicators before insert/overwrite
  • Disable unused tracks: Turn off auto-select for tracks you're not actively using
  • Audio follows video: Usually you want audio targeted to same relative position

‚ö†Ô∏è Common Mistakes:

  • Wrong track targeted: Always check target before pressing insert/overwrite
  • Audio not following: If video goes to V2, make sure audio targets A2 if needed
  • All tracks targeted: Can cause chaos - target only what you need
  • Locked track targeted: You'll get an error - unlock or change target
  • Forgetting to change target: Develop muscle memory for Alt/Opt + arrows

üéì Practice Exercises

Exercise 1: Track Targeting Basics (10 minutes)

  1. Create a timeline with 4 video tracks and 4 audio tracks
  2. Practice targeting each track using Alt/Opt + arrows
  3. Verify the target indicators update correctly
  4. Insert a test clip on different tracks
  5. Goal: Confidently target any track without looking

Exercise 2: B-Roll Workflow (15 minutes)

  1. Place an interview clip on V1 + A1
  2. Target V2, disable A1
  3. Insert 5 B-roll clips over the interview
  4. Target V1 again and make a cut in the interview
  5. Verify B-roll on V2 wasn't affected

Exercise 3: Multi-Track Organization (20 minutes)

  1. Set up a complex timeline:
    • V1: Main edit
    • V2: B-roll
    • V3: Graphics/titles
    • A1-A2: Dialogue
    • A3-A4: Music
  2. Practice switching targets to add content to each track type
  3. Lock tracks as you finish each one
  4. Goal: Smooth workflow without mis-targeted edits
