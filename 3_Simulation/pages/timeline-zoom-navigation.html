üí° Pro Tips for Efficient Navigation

Speed Up Your Workflow:

  • Learn the H key: The hand tool (H) is your best friend for fast panning
  • Use Shift+Z constantly: Getting the full overview helps maintain context
  • Zoom progressively: Don't jump from max zoom out to max zoom in
  • Follow the playhead: Enable playhead-centered scrolling during playback
  • Use markers as anchors: Place markers at key sections for quick navigation
