What are Coarse & Fine Dials?

In DaVinci Resolve, coarse and fine dials provide two levels of precision control for color grading and other adjustments. Coarse adjustments make large, quick changes, while fine adjustments provide precise, incremental control for detailed work. These are prominently featured in the Inspection page for detailed color correction.

ğŸ” Inspection Page Usage

The Inspection page is where coarse and fine dials shine in DaVinci Resolve. Located in the Color page, the Inspection page provides detailed control over:

  • RGB Balance: Adjust red, green, and blue channels individually
  • Lift, Gamma, Gain: Fine-tune shadows, midtones, and highlights
  • Temperature & Tint: Precise white balance correction
  • Hue, Saturation, Luminance: Detailed color adjustments

Coarse and fine controls allow you to make both broad strokes and surgical adjustments in the same workspace.

âš–ï¸ Coarse vs Fine Adjustments

ğŸ”¶ Coarse Adjustments

Step Size: Large increments (e.g., 5-10 units)

Speed: Fast, broad changes

Use When:

  • Initial color correction
  • Rough exposure adjustments
  • Quick white balance fixes
  • Getting close to target values

Example: Quickly bring down overexposed highlights from 100 to 50

ğŸ”· Fine Adjustments

Step Size: Small increments (e.g., 0.1-1 unit)

Speed: Slow, precise control

Use When:

  • Final polish and refinement
  • Matching shots precisely
  • Subtle skin tone adjustments
  • Achieving exact target values

Example: Tweaking skin tone from 49.8 to 50.2 for perfect match

ğŸ¯ When to Use Each Type

Coarse Dial Workflow:

  1. Initial Assessment: Use coarse to quickly survey the adjustment range
  2. Rough Correction: Get values into the ballpark with large movements
  3. Speed Over Precision: When exact values aren't critical yet
  4. Exploration: Quickly test different looks and settings

Typical Workflow: Start with coarse â†’ Get close â†’ Switch to fine â†’ Perfect it

Fine Dial Workflow:

  1. Final Refinement: After coarse adjustments, polish with fine control
  2. Shot Matching: Match color/exposure between shots precisely
  3. Subtle Changes: Imperceptible adjustments that add up
  4. Target Values: Hit specific numeric targets exactly

Typical Workflow: Coarse done â†’ Fine-tune â†’ Compare â†’ Adjust â†’ Repeat

âŒ¨ï¸ Keyboard Modifiers for Precision

Modifier Keys (works with mouse wheel/trackpad):

  • No Modifier: Normal adjustment speed
  • Shift + Scroll: Coarse adjustments (faster, larger steps)
  • Alt/Opt + Scroll: Fine adjustments (slower, smaller steps)
  • Ctrl/Cmd + Scroll: Very fine adjustments (ultra-precise)
  • Double-click: Reset to default value
  • Right-click â†’ Reset: Reset individual parameter

ğŸ’¡ Pro Tips for Coarse & Fine Control

Professional Workflow Strategies:

  • Coarse first, always: Never start with fine adjustments - use coarse to get 80% there
  • Watch your scopes: Use waveform/vectorscope to guide adjustments
  • Make small fine moves: Fine adjustments should be barely visible individually
  • Compare constantly: Toggle before/after to verify fine adjustments are improvements
  • Use reference stills: Gallery stills help you judge if fine adjustments are working
  • Know your limits: If fine adjustments aren't working, go back to coarse
  • Keyboard shortcuts: Learn modifier keys to switch between coarse/fine without UI

ğŸ¬ Common Use Cases

1. Skin Tone Matching

Scenario: Matching actor's skin tone across two shots

Workflow:

  1. Coarse: Adjust gamma to get overall brightness close
  2. Coarse: Tweak tint/temperature to get color family right
  3. Fine: Precisely match specific skin tone hue
  4. Fine: Adjust saturation in tiny increments
  5. Fine: Final luminance tweaks for perfect match

Result: Imperceptible skin tone match across cuts

2. White Balance Correction

Scenario: Fixing white balance on incorrectly shot footage

Workflow:

  1. Coarse: Temperature dial - swing warm/cool to find range
  2. Coarse: Tint dial - add green/magenta broadly
  3. Fine: Temperature - dial in exact neutral point
  4. Fine: Tint - remove any color cast

Result: Perfectly neutral whites and grays

3. Exposure Recovery

Scenario: Recovering overexposed highlights

Workflow:

  1. Coarse: Gain - pull down highlights significantly
  2. Coarse: Gamma - adjust midtones to compensate
  3. Fine: Gain - precisely recover just below clipping
  4. Fine: Gamma - maintain contrast without crushing

Result: Maximum highlight detail without artifacts

âš ï¸ Common Mistakes:

  • Starting with fine: Wastes time - always start coarse
  • Over-adjusting in fine mode: If you're making huge fine adjustments, switch to coarse
  • Ignoring the reset: If you've gone too far, reset and start over
  • Not using reference: Fine adjustments are meaningless without comparison
  • Forgetting mode: Check if you're in coarse or fine mode before adjusting

ğŸ“ Practice Exercises

Exercise 1: Coarse to Fine Workflow (10 minutes)

  1. Open a clip in the Color page â†’ Inspection mode
  2. Make a drastic change using coarse controls (e.g., make it very blue)
  3. Use coarse to get it back to roughly normal
  4. Switch to fine and perfect the adjustment
  5. Goal: Feel the difference in control precision

Exercise 2: Shot Matching (15 minutes)

  1. Select two shots with slightly different white balance
  2. Use coarse controls to get them in the same ballpark
  3. Switch to fine mode for all controls
  4. Make tiny adjustments until they match perfectly
  5. Toggle between shots to verify imperceptible transition

Exercise 3: Stream Deck+ Dial Practice (20 minutes)

  1. Configure Stream Deck+ dials for Lift/Gamma/Gain
  2. Practice switching modes by pressing the dial
  3. Make coarse adjustments with rotation
  4. Toggle to fine mode and refine
  5. Goal: Muscle memory for mode switching and control feel
