üìã Prerequisites

  • Windows 10 or Windows 11 (plugin is Windows-only)
  • Stream Deck hardware (any model: Mini, Classic, XL, or +)
  • Stream Deck software v6.0 or later installed and running
  • Elgato account (for Stream Deck Store access)
  • At least one additional application to manage (e.g., browser, editor, terminal)
  • Multi-monitor setup recommended for best results (but not required)
‚ö†Ô∏è Note: The Windows Mover plugin is Windows-only. macOS and Linux users should look at alternative tools like Rectangle (macOS) or i3 tiling (Linux).

üõ†Ô∏è Steps to Implement

Phase 1: Install the Plugin

  1. 1
    Open Stream Deck Software
    Launch the Elgato Stream Deck application from your system tray or Start menu.
  2. 2
    Open the Stream Deck Store
    Click the More Actions... button at the bottom-right of the Stream Deck app, or click the store icon in the top bar.
  3. 3
    Search for "Window Mover"
    In the Store search bar, type Window Mover or Windows Mover. You can also browse the System category.
  4. 4
    Install the Plugin
    Click Install on the Windows Mover plugin. The plugin will appear in your actions list under Custom or System.

Phase 2: Configure Window Positions

  1. 5
    Drag the Action to a Button
    From the actions panel, drag Window Mover onto an empty Stream Deck button.
  2. 6
    Set the Target Application
    In the action's settings, enter the application name or process (e.g., chrome, code, resolve). You can also use window title matching.
  3. 7
    Define Position & Size
    Set the target position: X, Y coordinates and Width, Height in pixels. Use preset positions like "Left Half", "Right Half", "Top-Left Quarter", etc., if available.
  4. 8
    Select Target Monitor
    If you have multiple monitors, choose which display the window should move to (Monitor 1, 2, 3, etc.).

Phase 3: Create a Layout Profile

  1. 9
    Create Multiple Buttons
    Set up one button per window position. For example: "Browser ‚Üí Left", "Editor ‚Üí Right", "Terminal ‚Üí Bottom".
  2. 10
    Add Custom Icons
    Assign recognizable icons to each button ‚Äî use the app icon + an arrow indicating direction for quick identification.
  3. 11
    Use Multi-Actions for Full Layouts
    Combine multiple Window Mover actions into a single Multi-Action button. One press can arrange your entire workspace: browser left, editor right, terminal bottom.
  4. 12
    Test & Fine-Tune
    Press each button and adjust X/Y/Width/Height values until windows land exactly where you want them. Note your screen resolution for reference.

üí° Pro Tips

üí° Tip: Create separate Stream Deck pages for different workflow layouts ‚Äî one for coding, one for video editing, one for meetings. Use Page Jump buttons to switch between them.
üí° Tip: Pair Window Mover with Multi-Actions to open an application AND position it in one button press. Add a small delay between actions for reliability.
üí° Tip: Use Windows display scaling settings to ensure pixel coordinates are consistent. If using 150% scaling, your effective resolution may differ from native.
