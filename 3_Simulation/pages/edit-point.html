What is an Edit Point?

An edit point is the precise moment in your timeline where you want to make a cut, insert footage, or perform an editing operation. It's the foundation of all video editing workflows and determines where clips are split, trimmed, or joined together.

üéØ Types of Edit Points

1. In Point (I)

The beginning of a clip selection. Marks where you want footage to start playing or where you want to begin an edit.

2. Out Point (O)

The ending of a clip selection. Marks where you want footage to stop playing or where you want to end an edit.

3. Playhead Position

The current position in your timeline. Used as a reference point for making cuts and inserting clips.

4. Cut Point

A position where you split a clip into two separate segments. Creates two edit points at once.

Essential Keyboard Shortcuts

  • I - Set In Point
  • O - Set Out Point
  • X - Mark Clip (sets both In and Out points)
  • Ctrl/Cmd + B - Blade/Cut at playhead (creates edit point)
  • Q - Trim to In Point
  • W - Trim to Out Point
  • Up/Down Arrow - Jump between edit points

üé¨ Edit Point Workflows

1. Three-Point Editing

The most common editing technique using edit points:

  1. Mark your source: Set In and Out points on the clip in your source viewer
  2. Mark your timeline: Position playhead or set In point in timeline
  3. Perform edit: Insert, overwrite, or replace at the edit point

2. Ripple Editing

Edit one side of an edit point while automatically adjusting the timeline to prevent gaps:

  • Trim from the edit point without leaving black gaps
  • All subsequent clips move to accommodate the change
  • Maintains sync and timeline continuity

3. Roll Editing

Adjust an edit point by simultaneously trimming the end of one clip and the start of the next:

  • Move the edit point between two clips
  • Maintains total timeline duration
  • Useful for fine-tuning cut timing

üí° Pro Tips for Working with Edit Points

  • Use markers: Place markers at potential edit points during review
  • Zoom in: Get frame-accurate precision when setting edit points
  • Snap to edit points: Enable timeline snapping to quickly align to existing edit points
  • Audio waveforms: Use audio waveforms to visually identify natural edit points
  • J-K-L editing: Combine playback controls with edit points for faster workflow

üîß Edit Point Best Practices

‚ö†Ô∏è Common Mistakes to Avoid

  • Cutting on movement: Edit points work best during moments of stillness
  • Ignoring audio: Always consider audio when placing edit points
  • Too many cuts: Not every moment needs an edit point
  • Forgetting to clear markers: Old In/Out points can cause confusion
  • Not using snapping: Enable snap for precise edit point alignment

Industry Standards:

  1. Action on cut: Place edit points where there's on-screen action or movement
  2. Audio bridge: Let audio carry across edit points for smoother transitions
  3. Motivated cuts: Edit points should have a reason (new angle, emphasis, pacing)
  4. L-cuts and J-cuts: Offset audio and video edit points for natural flow
  5. Beat and rhythm: Place edit points to match music or natural pacing

üìä Edit Point Terminology Reference

Hard Cut

Direct transition from one clip to another at the edit point with no transition effect.

Soft Cut

Edit point with a short dissolve or transition to smooth the change.

Jump Cut

Edit point within the same shot that creates a noticeable "jump" in time.

Match Cut

Edit point that connects two similar compositions or actions across different shots.

L-Cut

Audio from the first clip extends past the video edit point into the next clip.

J-Cut

Audio from the second clip starts before the video edit point.
