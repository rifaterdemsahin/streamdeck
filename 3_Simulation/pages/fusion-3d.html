‚öôÔ∏è Setup Guide

Step 1: Install Fusion Page Plugin

  1. Open Stream Deck software
  2. Navigate to More Actions ‚Üí DaVinci Resolve
  3. Install the official DaVinci Resolve plugin if not already installed
  4. Drag Fusion-specific actions to your Stream Deck layout

Step 2: Configure Rotary Dials

  1. Click on the dial configuration in Stream Deck software
  2. Set dial actions to "Adjust Value" or custom keyboard shortcuts
  3. Map to Fusion inspector parameters (X, Y, Z positions)
  4. Set rotation sensitivity (recommended: 0.1 - 1.0 units per click)
  5. Configure push-button action to reset or toggle modes

Step 3: Link to Fusion Inspector

In DaVinci Resolve Fusion page:

  1. Select a 3D node (Transform 3D, Camera, Light, etc.)
  2. Open the Inspector panel (Ctrl/Cmd + I)
  3. Right-click on parameter (e.g., "X Position")
  4. Select "Assign to MIDI Controller" or use keyboard mapping
  5. Test dial rotation to ensure smooth parameter control

Step 4: Create Profiles for Different Workflows

  • Camera Profile: Focus on camera movement and positioning
  • Particle Profile: Emitter controls, velocity, gravity
  • Lighting Profile: Light intensity, color, position
  • Transform Profile: Object rotation, scale, position

üéØ Dial Sensitivity Settings

  • Position dials: 1-5 units per rotation step (for smooth movement)
  • Rotation dials: 1-15 degrees per step (for precise angles)
  • Zoom/Scale: 0.05-0.1 units per step (for gradual changes)
  • Intensity: 0.01-0.05 per step (for subtle lighting adjustments)
