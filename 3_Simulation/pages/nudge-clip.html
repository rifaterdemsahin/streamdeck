üõ†Ô∏è Setup Steps

Phase 1: Set DaVinci Resolve Keyboard Shortcuts

  1. 1
    Open Keyboard Customization
    In DaVinci Resolve, go to DaVinci Resolve ‚Üí Keyboard Customization (or Ctrl+Alt+K on Windows).
  2. 2
    Find Nudge Commands
    Search for Nudge in the search bar. You'll see:
    ‚Ä¢ Nudge One Frame Left ‚Äî default: , (comma)
    ‚Ä¢ Nudge One Frame Right ‚Äî default: . (period)
  3. 3
    Verify or Assign Shortcuts
    Make sure the shortcuts are assigned. If not, assign , for nudge left and . for nudge right (these are DaVinci Resolve defaults).

Phase 2: Configure the Stream Deck + Knob

  1. 4
    Select a Dial Slot
    Open Stream Deck software ‚Üí select your Stream Deck + ‚Üí click on a dial slot (e.g., Knob 3).
  2. 5
    Assign Rotate CW (Clockwise)
    Set the Rotate Right action to send the hotkey . (period) ‚Äî this triggers "Nudge One Frame Right" in Resolve.
  3. 6
    Assign Rotate CCW (Counter-Clockwise)
    Set the Rotate Left action to send the hotkey , (comma) ‚Äî this triggers "Nudge One Frame Left" in Resolve.
  4. 7
    Assign Click (Optional)
    Set the Press action to Ctrl+Z (Undo) ‚Äî quickly undo a nudge if you went too far.

Phase 3: Label & Test

  1. 8
    Label the Touch Strip
    Set the LCD touch strip below the knob to display ‚Üî NUDGE so you always know which knob does what.
  2. 9
    Test in DaVinci Resolve
    Open a project ‚Üí select a clip on the timeline ‚Üí rotate the nudge knob slowly. Each detent (click) should move the clip exactly 1 frame left or right.
  3. 10
    Fine-Tune the Sensitivity
    If the knob sends multiple nudges per rotation, adjust the dial sensitivity in Stream Deck settings. You want 1 event per detent for precise control.

üí° Pro Tips

üí° Tip: Hold Shift while nudging to move the clip by 5 frames at a time instead of 1. Map a button above the nudge knob to toggle Shift for quick switching.
üí° Tip: Select multiple clips before nudging to move them all together ‚Äî great for sliding an entire sequence left or right while keeping relative timing intact.
üí° Tip: Combine the nudge knob with the jog knob ‚Äî use jog to find the right frame, then switch to the nudge knob to slide the clip into position.
