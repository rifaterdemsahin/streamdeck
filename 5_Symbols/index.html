Context-Aware Rules — Stay in Flow
🎬 Video Production Day
📅Sunday script write → AI Flow profile. Monday–Friday editing → Video profile auto-activates when Resolve opens.
🎯7-day pipeline: script Sun → record Mon → edit Tue–Thu → review Fri → publish Sat.
🤖Dial 4 rotates AI context: Claude → DeepSeek → fal.ai → Remotion render.
🎙️ElevenLabs VO key: auto-uses selected Resolve subtitle text as input.
💡Marker AI key: reads current clip → asks Claude to generate section title + tags.
⚙️ SRE / Contract Day
🚨Incident key opens runbook + posts to Teams + starts timer automatically.
📊Loki key launches pre-built LogQL query for active namespace (reads from clipboard).
🔐Vault key auto-renews token and copies to clipboard — no manual auth loop.
🏷️Feature flag key opens YAML + runs AI diff comparison against last deploy.
🔄Rollback key requires double-press confirmation to prevent accidents.
📋 Contract / Dispute
⚖️ACAS tracker key shows days elapsed since EC notification, next deadline.
✉️Solicitor draft key opens Claude with pre-loaded context: chain of parties, amounts, dates.
📂Evidence folder key timestamped auto-screenshot capture → named evidence archive.
⏱️Timesheet key auto-fills current week hours from calendar entries.
🤝MSP chain key shows visual map: Contract → MSP1 → MSP2 → Agency1 → Agency2 → Umbrella.
🏠 Family Mode
⏰Auto-pulsing at 08:00 (morning school run) and 15:15 (afternoon pickup). Mira & Arya alert.
🌐Net unblock key → Telegram message to n8n → EdgeRouter firewall rule lifted for 30min.
💑Ece quick message: pre-typed templates ("On my way", "In a call", "5 mins").
🏡Home lab key: Proxmox VM status + Pi Zero 2W HID bridge health check.
📵Family mode dims all contract/SRE keys to prevent accidental actions.
🤖 AI Deep Work / Flow
🧠All notification keys disabled. Mic mute auto-on. DND mode active system-wide.
📎Clip→AI key: reads clipboard → infers intent → routes to correct AI (Claude/DeepSeek/local).
⏲️Pomodoro key enforces 25/5 rhythm. After 4 rounds → 30min break alarm.
🗂️Obsidian capture: AI summarises active window content → appends to daily note.
✍️Script write key: loads current week's YouTube script template + brief into Claude.
